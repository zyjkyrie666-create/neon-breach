# 霓虹突围（Neon Breach）

霓虹突围是一款基于 HTML5 Canvas 开发的零依赖 2D 生存射击游戏。项目包含完整的战斗循环、敌人 AI、动态波次、碰撞检测、对象池、实时性能面板和可视化关卡编辑器。

## 在线功能

- 三类敌人：追击型无人机、远程射手和重装单位。
- 战斗系统：移动、瞄准、连续射击、冲刺无敌帧、生命值、分数与连击倍率。
- 动态波次：敌人数量、生命值和移动速度随波次逐步提升。
- 关卡系统：支持墙体碰撞、多个敌人出生点和自定义玩家出生点。
- 关卡编辑器：可视化编辑地图，支持 JSON 导入导出、自动保存和一键测试。
- 性能优化：固定时间步、对象池和空间哈希碰撞宽阶段。
- 调试工具：按 `F3` 查看 FPS、活动对象数量和碰撞候选数量。
- 纯代码资源：画面、粒子、光效和声音均由代码实时生成。

## 技术实现

- 使用 `requestAnimationFrame` 驱动渲染，固定时间步更新游戏逻辑。
- 使用 Canvas 坐标变换、旋转、渐变与混合模式实现霓虹视觉效果。
- 使用对象池复用子弹、敌人与粒子，减少高频创建和回收对象带来的开销。
- 使用空间哈希划分碰撞区域，减少墙体碰撞检测的候选数量。
- 使用二维数组保存网格关卡，通过 JSON 完成地图数据的导入与导出。
- 使用 `localStorage` 保存最高分和自定义关卡。

## 运行方法

项目不需要安装第三方依赖。在项目根目录启动静态服务器：

```powershell
python -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```

也可以直接打开 `index.html`，但使用本地静态服务器更接近实际部署环境。

## 操作说明

- `W/A/S/D`：移动
- 鼠标移动：瞄准
- 鼠标左键或 `J`：射击
- `Space`：冲刺，冲刺期间免疫伤害
- `Esc`：暂停或继续
- `F3`：显示或隐藏性能面板
- 游戏结束后按 `R`：重新开始

## 关卡编辑器

切换到“关卡编辑”模式后，可以放置墙体、敌人出生点和玩家出生点，也可以删除已有元素。编辑结果会自动保存在浏览器本地存储中。

编辑器支持：

1. 导出当前关卡 JSON。
2. 载入已有的关卡 JSON。
3. 将地图恢复为默认布局。
4. 直接测试当前编辑的关卡。

## 测试方法

1. 启动项目并点击“启动作战”。
2. 检查移动、瞄准、射击和冲刺是否正常。
3. 消灭第一波敌人，检查分数、连击和波次提示是否更新。
4. 按 `F3`，检查 FPS、敌人、子弹、粒子和碰撞候选数是否实时变化。
5. 进入“关卡编辑”，放置墙体和敌人出生点后测试当前关卡。
6. 刷新页面，检查自定义关卡和最高分是否仍然保留。

## 项目描述

- 独立开发基于 HTML5 Canvas 的 2D 生存射击游戏，实现角色控制、敌人 AI、武器、碰撞、动态波次、连击与本地存档等核心系统。
- 设计数据驱动的网格关卡格式与可视化编辑器，支持墙体和出生点编辑、JSON 导入导出、自动保存及一键测试。
- 使用固定时间步保证逻辑稳定，通过对象池复用高频实体，并使用空间哈希减少碰撞宽阶段候选数量。
- 实现实时性能面板，用于观察 FPS、活动对象和碰撞候选数量，验证运行状态与优化效果。
- 使用纯代码完成角色、场景、粒子、光晕、屏幕震动和合成音效，无需第三方依赖或外部美术资源即可运行。

## 浏览器说明

合成音效会在首次点击“启动作战”后启用，这是浏览器自动播放策略的要求。
